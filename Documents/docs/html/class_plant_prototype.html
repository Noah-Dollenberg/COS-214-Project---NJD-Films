Detailed Description

Abstract prototype class for cloning plants.

This class implements the Prototype design pattern, allowing plants to be cloned/propagated without knowing their concrete types. This is useful for creating new plants from existing specimens through vegetative propagation (cuttings, division, etc.). Each concrete prototype knows how to clone itself.
